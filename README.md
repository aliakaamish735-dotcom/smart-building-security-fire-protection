# Smart Building Security and Fire-Protection System

Welcome to the repository.

## Repository Contents

### 📁 `code/`

Contains the Arduino source code, firmware, diagnostics, and calibration
files.

### 📁 `data/`

Contains datasets, calibration files, CSV logs, and Excel workbooks.

### 📁 `docs/`

Contains the proposal, presentations, architecture diagrams, Fritzing
design, component documentation, and individual contributions.

### 📁 `report/`

Contains the final technical report.

### 📁 `video/`

Contains project demonstration and presentation videos.
