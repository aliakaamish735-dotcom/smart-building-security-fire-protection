# Project Documentation

This folder contains the supporting documentation for the **Smart
Building Security and Fire-Protection System**.

## Files Included

### `Project_Team_Members.pdf`

Lists the project team members and their student IDs.

### `README_components.md`

Describes the hardware components, sensors, actuators, power supplies,
wiring accessories, and materials used in the project.

### `Smart_Laboratory_Security_Fire_Protection_Updated.pptx`

Final presentation used for the 10-minute technical presentation.

### `architecture_image.jpeg`

Illustrates the overall architecture of the system and the interaction
between the security and fire-protection subsystems.

### `questions&answers_ppt.pptx`

Contains the expected questions and prepared answers for the project
presentation and discussion.

### `sensors_architecture1.fzz`

Final Fritzing project containing the complete hardware wiring and
breadboard design.

### `sensors_project_proposal.pdf`

Original project proposal describing the objectives, requirements,
implementation plan, deliverables, and evaluation criteria.

## Purpose

These documents provide the technical background, hardware
documentation, presentation material, system architecture, and project
planning used throughout the design and implementation of the Smart
Building Security and Fire-Protection System.
